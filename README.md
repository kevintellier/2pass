# 2pass

## Install

```
pip3 install -r requirements.txt
```

## Usage

```
python3 ./2pass.py
```

Help :

2pass [-f inputFile] [-o inputFile] [-i passwordID] [-h] command
    -f Vault file containing passwords
    -o Output vault inputFile name
    -i Password ID
    -h Display help
Commands
ls    add     rm    create
